# 카페 BGM 오디오 파일

이 폴더에 저작권 없는 카페용 BGM MP3 파일을 넣어 주세요.

## 추천 무료 음원 사이트 (저작권 없음/무료 사용)

- **[Pixabay Music](https://pixabay.com/music/search/cafe/)** – 무료, 저작권 없음, 출처 표시 불필요
- **[Mixkit](https://mixkit.co/free-stock-music/jazz/)** – 무료 재즈/라운지 음악
- **[Incompetech](https://incompetech.com/music/royalty-free/?genre=Jazz)** – CC BY (출처 표기 필요)

## 사용 방법

1. 위 사이트에서 카페/재즈/라운지 계열 음악을 다운로드
2. MP3 파일을 이 폴더에 저장 (예: `cafe1.mp3`, `cafe2.mp3`)
3. `components/coffee-ambience.tsx`의 `CAFE_BGM_PLAYLIST` 배열 경로가 이미 설정되어 있음 (`/audio/cafe1.mp3` 등)
4. 파일을 추가하면 바로 재생됩니다
